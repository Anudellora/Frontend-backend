Заметки — PWA Приложение

Версия 2.0 • App Shell Architecture

🏗️

App Shell Architecture

Приложение построено по архитектуре App Shell. Статическая оболочка (навигация, шапка, стили) кэшируется при первой загрузке. Динамический контент страниц загружается по стратегии Network First.

📴

Офлайн режим

Все статические ресурсы (HTML-оболочка, CSS, JS, иконки) кэшируются при установке Service Worker. Динамические страницы сначала запрашиваются из сети, а при отсутствии соединения — обслуживаются из кэша.

🔒

HTTPS

Приложение работает по протоколу HTTPS с использованием самоподписанного сертификата. Это необходимо для корректной работы Service Worker и обеспечения безопасности данных.

💾

Локальное хранилище

Заметки хранятся в localStorage браузера. Это позволяет сохранять данные между сессиями без серверной части. При офлайн-режиме все данные остаются доступными.

📱

PWA (Progressive Web App)

Приложение можно установить на рабочий стол устройства. Оно работает как нативное приложение благодаря Web App Manifest и Service Worker.

⚡

Стратегии кэширования

Cache First — для статических ресурсов App Shell (мгновенная загрузка).
Network First — для динамических страниц (актуальные данные, с фоллбэком на кэш).

Технологии

HTML5 CSS3 JavaScript ES6+ Service Worker API Cache API Web App Manifest localStorage HTTPS (TLS)
